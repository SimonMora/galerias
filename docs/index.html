Galerías

Un proyecto de microservicios que contiene Eureka Server

¿Cómo funciona?

Imagen API

Es un microservicio que gestiona la base de datos de imágenes.

Galería API

Microservicio que tiene implementado un cliente para facilitar la comunicación con Imágenes API.

Eureka Server con Ribbon

Es el servidor donde están declarados todos los microservicios. Además incluye Ribbon: un servicio de balanceo de cargas que facilita el consumo de APIs en la aplicación.

Zuul Gateway Server

Es un microservicio que gestiona las peticiones y las respuestas del sistema.

Authentication con JWT

Se encarga de asegurar las peticiones que se puedan realizar, utilizando tokens.

Zuul interpreta y gestiona las peticiones que se puedan realizar desde el Front-End de la aplicación, actuando de intermediario con Eureka Server.
